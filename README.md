# AI Interview Copilot

An AI-powered interview preparation platform built with the MERN Stack and Google Gemini API.

## Features
- AI Interview Report
- Resume Upload
- Authentication
- Dashboard
- Responsive UI

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- Google Gemini API
- Tailwind CSS v4 latest

## Installation

### Frontend
npm install
npm run dev

### Backend
npm install
npm run dev

## Environment Variables

Create a .env file in Backend:

PORT=
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=
